HTML5 Geolocation

What is this?

Explore the current google search trends around the world, that is, at least the countries for which there are data! Click on a marker to show a list of the first search trends in the country. Click again or on the "x" to close the popup.

This app functions thanks to the google trends api from which the data are downloaded. Data are then displayed in an animated world map using country coordinates. The world map is powered by webGL Earth.
